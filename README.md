This is a personal portfolio website built with Next.js, TypeScript and Tailwind CSS.

## What this repo contains

- A Next.js app under the `app/` directory that implements pages and layout.
- Design system and UI primitives under `components/` and `components/ui/`.
- Global styles in `app/globals.css`.

## Tech stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion / GSAP (animations)

## Local development (Windows - cmd.exe)

1. Install dependencies (this project uses pnpm):

   pnpm install

2. Start the development server:

   pnpm dev

3. Open the site in your browser:

   http://localhost:3000

## Production build

1. Build:

   pnpm build

2. Start production server:

   pnpm start
